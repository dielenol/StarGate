## 설명
항생제는 NOSB S1E2 "선택" 작전 보상으로 과학자 직군에 해금된 감염 대응 계열 소모품이다. 오디세이 시설 사건 이후 감염, 병원체, 기생성 전염 위험에 대응하기 위한 후방 지원 품목으로 분류한다.

## 배경
화이트로즈 측은 작전 종료 후 노부스 오르도 현장팀에 협력 의사를 전했고, 사무국은 해당 작전 보상 목록에 과학자 직군 항생제 해금을 기록했다.

## 획득 경로
S1E2 선택 작전 보상으로 해금된다. 실제 지급 수량, 제조 조건, 상점 재고 반영 여부는 별도 승인 대상이다.

## 비고
세부 회복량, 감염 단계 완화 수치, 사용 조건은 현재 보상 기록만으로 확정하지 않는다. 이 문서는 해금 사실과 카탈로그 연결을 위한 기준 기록이다.
